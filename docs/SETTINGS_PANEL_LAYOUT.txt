INTERACTION FLOWS:
==================

1. CHANGING HOTKEY:
   User types in hotkey field → clicks "Aplicar" → 
   Validation → Save to config → Success message

2. CHANGING THEME:
   User selects theme from dropdown →
   Auto-save → Panel theme updates → Success message

3. TOGGLING MODULE:
   User clicks checkbox →
   Auto-save → Success (no restart yet)

4. EXPORTING CONFIG:
   User clicks "Exportar" → File dialog opens →
   User chooses location → Save → Success message

5. IMPORTING CONFIG:
   User clicks "Importar" → File dialog opens →
   User chooses file → Confirmation dialog →
   Import → Validate → Save → Refresh UI → Success

6. RESTARTING SERVICES:
   User clicks "Reiniciar Servicios" →
   Reload config → Restart indexer/AI/modules →
   Show status dialog with results

STYLING:
========

Dark Theme:
- Background: #1c1c1c
- Text: #eaeaea
- Group Background: #252525
- Border: #3a3a3a
- Buttons: Accent color (#3a86ff by default)
- Button hover: Slightly transparent
- Button pressed: More transparent

- Background: #f5f7fb
- Text: #0f172a
- Group Background: #ffffff
- Border: #cbd5e1

RESPONSIVE BEHAVIOR:
====================

- Panel is scrollable for small screens
- Minimum width: 600px
